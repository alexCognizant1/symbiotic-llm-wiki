# Symbiotic LLM Wiki

**A symbiotic, LLM-agnostic implementation of Karpathy's LLM Wiki pattern in Obsidian.**

Focused on deep research in:
- Category Theory
- Functional Programming & Computer Languages
- Agentic AI & Memory Systems
- Mendix (low-code + agentic development)
- AI Research

## Core Philosophy

This is **not** about full automation or strict human control.

It is about **symbiosis**:
- You provide direction, taste, semantic grounding, and judgment.
- The system provides breadth, cross-connection, recall, formalization attempts, and structural insight.
- Both get better over time through co-evolution.
- The wiki is the **shared external mind** — persistent, inspectable, and versioned.

## Quick Start

1. Clone this repo
2. Open the folder in Obsidian
3. Read `WIKI-PROTOCOL.md` (the contract/schema)
4. Start adding sources to `raw/`
5. Use your agent (Claude Code, Cursor, etc.) with the protocol
